# UI Studio

A visual builder for creating React applications using declarative JSON specifications.

## ✅ Milestone 1: Core & Persistence - COMPLETE

### Completed Features

✅ **Project Setup** - TypeScript, Vite, ESLint, Prettier, React 18
✅ **Schema Definitions** - All types defined in [src/core/schema/](src/core/schema/)
✅ **Validation** - Zod schemas with naming/reference/performance checks
✅ **IndexedDB Adapter** - Full CRUD operations with search and sorting
✅ **Locking Mechanism** - Cross-tab coordination with BroadcastChannel
✅ **Migration Framework** - Sequential schema versioning system
✅ **Home Screen UI** - Create, view, duplicate, and delete projects

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
